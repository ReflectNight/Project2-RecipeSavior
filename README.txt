DESCRIPTION
=====================
A recipe book.


=====================
HOW-TO
=====================
Recipe Savior is a way to store and share recipes. 

It can be used to create new recipes by clicking the "New" button, viewing recipes using the "View" button, editing recipes using the "Edit" button, deleting recipes using the "Delete" button, and searching for recipes using the "Search" button.
